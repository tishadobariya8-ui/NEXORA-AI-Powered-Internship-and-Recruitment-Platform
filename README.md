# 🚀 NEXORA – AI Powered Internship Platform

NEXORA is a full-stack AI-powered internship platform that connects students with companies and startups. It helps students discover internships, apply online, analyze resumes using AI, compare resumes with job descriptions, generate cover letters, and practice AI-generated interview questions.

---

## ✨ Features

### 👨‍🎓 Student
- User Registration & Login
- Complete Profile
- Upload Resume (PDF)
- Browse Internships
- Apply for Internships
- View Applied Internships
- Dashboard Overview

### AI Career Assistant
- ATS Resume Analyzer
- Job Match Score
- AI Cover Letter Generator
- AI Interview Questions with Ideal Answers

### Internship Management
- Internship Listings
- Internship Details
- Application Tracking
